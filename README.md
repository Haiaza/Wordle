# Wordle

Wordle is a word guessing game.  

![Logo]((image.png))

# Technologies Used
HTML - CSS - JS

# How to Play
Think of a 5-letter word, and try your luck.
The color change indicates how good your guess was.

Green - Perfect!
Yellow - Right idea, wrong neighborhood
Gray - Wrong
